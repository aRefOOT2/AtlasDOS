## Blasn Projectとは

**Blasn Project**とは、個人で開発しているOS(CUI OS、GUI OS)・カーネルのことです。

主に、

- Blasn Kernel

- Blasn Kernel Shell

- Blasn OS

が該当します。

## Blasn OSとは

**Blasn OS**は、個人が開発しているOSの名称です。Blasn OSの実行可能ファイルは「Blasn eXecutable Object」です。Blasn OSは主にCUIでの操作になりますが、今後はGUIバージョンも開発する予定です。
